如果你曾经在中型或大型的 React 应用程序中工作过，你可能注意到组件会频繁重新渲染。在某些情况下，这可能导致性能问题，应用程序顶层的小变化会导致大量不相关的组件重新渲染。

在这一组课程中，我们将深入学习 React 组件为何重新渲染的原因。我们还将看到三个 React 工具如何帮助我们优化这些内容：

- React.memo 组件包装器
- useCallback 钩子
- useMemo 钩子
